Even More Features

Cloud/S3 Storage

Cloud native from the start. You can choose/cofigure it to store all documents in S3 storage.

Multi User

Papermerge DMS is a multi user system. You can manage rights and permissions per user.

Highly Scalable

It does not matter how many documents you want to OCR, store and index. Papermerge is highly scalable.

REST API

Want to import documents from SFTP account? Or maybe import all your EMail attachments? REST API is always there to assist you.

Receipts are Welcome

It can handle long paper receitps as well as standard A4 paper receitps.

Scanners Best Friend

It is a natural addition to any scanner. It supports most common scanner output formats - PDF, JPEG, PNG or TIFF

The most amazing part of the whole project is that there is a great community and great documentation around it.
